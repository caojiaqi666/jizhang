## 🍎 iPhone 用户安装步骤

### 第1步：用 Safari 打开网址

```
⚠️ 重要：必须使用 Safari 浏览器！
其他浏览器（Chrome、微信内置浏览器等）不支持
```

在 Safari 中输入网址：`你的网址`

### 第2步：点击分享按钮

找到底部工具栏的**分享按钮**（一个向上的箭头 📤）

<img src="https://via.placeholder.com/300x60/14b8a6/ffffff?text=%E5%88%86%E4%BA%AB%E6%8C%89%E9%92%AE" alt="分享按钮">

### 第3步：选择"添加到主屏幕"

在弹出的菜单中，**向下滚动**，找到"添加到主屏幕"

<img src="https://via.placeholder.com/300x400/10b981/ffffff?text=%E6%B7%BB%E5%8A%A0%E5%88%B0%E4%B8%BB%E5%B1%8F%E5%B9%95" alt="添加到主屏幕">

### 第4步：确认添加

1. 可以修改应用名称（默认是"麻薯记账"）
2. 点击右上角的**"添加"**按钮
3. 完成！🎉

### 第5步：开始使用

返回主屏幕，你会看到麻薯记账的图标 🍡

点击图标就可以使用了，和普通 APP 一样！

---

## 🤖 Android 用户安装步骤

### 第1步：用 Chrome 打开网址

```
建议使用 Chrome 浏览器
其他基于 Chromium 的浏览器（Edge、Samsung Internet）也可以
```

在 Chrome 中输入网址：`你的网址`

### 第2步：等待安装提示

Chrome 会自动弹出一个安装提示：

```
┌─────────────────────────────┐
│  麻薯记账                     │
│  要将此应用添加到主屏幕吗？    │
│                              │
│  [取消]  [安装] ←点这里      │
└─────────────────────────────┘
```

### 第3步：点击"安装"

点击提示框中的**"安装"**按钮

### 第4步：完成！

应用会被添加到：
- 📱 主屏幕
- 📋 应用抽屉（所有应用列表）

就像从应用商店下载的 APP 一样！

### 如果没有看到提示？

点击浏览器右上角的菜单（⋮），选择：
- "添加到主屏幕"
- 或 "安装应用"

## 💻 电脑用户安装步骤

### Windows/Mac/Linux

1. 使用 **Chrome** 或 **Edge** 浏览器访问网址
2. 在地址栏右侧会出现一个**安装图标** ⊕
3. 点击安装图标
4. 在弹出的对话框中点击**"安装"**
5. 完成！

安装后：
- 应用会作为独立窗口打开
- 可以固定到任务栏/Dock
- 可以在开始菜单/应用列表中找到

## ❓ 常见问题

### Q: 需要付费吗？
**A: 完全免费！** 无需应用商店，无需付费。

### Q: 会占用多少空间？
**A: 非常小！** 通常只有 5-10 MB，远小于普通 APP。

### Q: 可以卸载吗？
**A: 可以！** 
- **iPhone**: 长按图标 → 删除 App
- **Android**: 长按图标 → 卸载
- **电脑**: 在设置中卸载（和普通软件一样）

### Q: 会自动更新吗？
**A: 是的！** 每次打开应用都会自动检查更新，无需手动更新。

### Q: 离线能用吗？
**A: 可以！** 已经查看过的数据可以离线访问，新数据需要联网同步。

### Q: 和从应用商店下载的有什么区别？
**A: 几乎没有区别！** 
- ✅ 同样的功能
- ✅ 同样的体验
- ✅ 同样的图标
- ✅ 但是完全免费，无需通过应用商店

### Q: 为什么 iPhone 上没有自动安装提示？
**A: 这是 iOS 的限制。** iPhone 用户需要手动"添加到主屏幕"，但只需要操作一次。

### Q: 数据安全吗？
**A: 安全！** 
- 使用 HTTPS 加密传输
- 数据存储在云端
- 和普通 APP 一样安全

### Q: 可以在多个设备上使用吗？
**A: 当然可以！** 数据会自动同步，在任何设备上都能访问。
